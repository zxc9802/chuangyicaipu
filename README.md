# 创意食谱 H5

一款趣味性 H5 应用，用户可输入任意食材、佐料及烹饪方式，AI 将智能推断菜品味道，给出星级评分，并生成菜品成品效果图，帮助用户探索烹饪创意。

## ✨ 核心功能

### 🔐 用户系统
- 手机号注册/登录
- 个人中心管理
- 历史记录查看

### 🥗 食材输入
- 支持多种食材输入
- 佐料及用量添加（如盐 5 克、酱油 10 毫升）
- 烹饪方式选择（炒、煮、炸、蒸、烤等）

### 🤖 AI 智能分析
- 推断菜品味道（鲜香微辣、酸甜可口等）
- 多维度综合评价（口味搭配、营养均衡、创意指数）
- 1-5 星级评分

### 🎨 成品图生成
- AI 自动生成菜品效果图
- 根据星级调整风格：
  - ⭐⭐⭐⭐⭐ 精致摆盘
  - ⭐⭐⭐ 简洁摆盘
  - ⭐⭐ 抽象动漫风格

### 📋 历史记录
- 自动保存创作记录
- 支持查看详情和删除

### 📸 打卡分享
- 一键生成打卡图片
- 包含食材、佐料、评价及成品图
- 支持下载分享

## 🛠️ 技术栈

- **前端框架**: React + TypeScript
- **构建工具**: Vite
- **样式方案**: TailwindCSS
- **后端服务**: Supabase
- **AI 能力**: 大模型文本分析 + AI 绘图
