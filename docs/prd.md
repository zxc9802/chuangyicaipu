# 创意食谱H5需求文档

## 1. 主题与简介
创意食谱是一款趣味性H5应用，用户可输入任意食材、佐料（含用量）及烹饪方式，AI将智能推断菜品可能的味道，给出专业评价，并生成菜品成品效果图，帮助用户探索烹饪创意。\n
## 2. 核心功能
### 2.1 食材输入模块
- 用户可输入多种食材（如鸡蛋、番茄、土豆等）\n- 支持添加佐料及用量（如盐5克、酱油10毫升、辣椒粉少许）\n- 可选择烹饪方式（如炒、煮、炸、蒸、烤等）\n
### 2.2 AI智能分析
- 根据用户输入的食材、佐料和烹饪方式，AI推断菜品可能呈现的味道（如鲜香微辣、酸甜可口等）\n- 对菜品进行综合评价，包括口味搭配合理性、营养均衡度、创意指数等维度

### 2.3 成品图生成
- AI绘图功能自动生成菜品成品效果图
- 根据AI评价结果调整图片风格：\n  + 评价较好时：生成精致摆盘的成品图，展示菜品的视觉美感
  + 评价不佳时：生成抽象风格的成品图，摆盘简化，视觉呈现更随意
- 展示菜品的视觉呈现效果

### 2.4 结果展示
- 统一展示AI推断的味道描述、评价内容及成品图
- 支持保存或分享结果

### 2.5 打卡图片生成
- 点击分享按钮，自动生成打卡图片
- 打卡图片内容包含：\n  + 使用的食材清单
  + 所用佐料及用量
  + 烹饪方式
  + AI给出的建议评价
  + 菜品成品图片
- 打卡图片支持下载功能

## 3. 设计风格
- 配色方案：以暖橙色为主色调（#FF8C42），搭配奶白色背景（#FFF8F0）和深灰色文字（#333333），营造温馨美食氛围
- 视觉细节：卡片采用8px圆角，添加轻微阴影（0 2px 8px rgba(0,0,0,0.1)）增强层次感；图标使用手绘风格，按钮带微动画反馈
- 整体布局：采用纵向卡片式布局，输入区、分析区、结果区依次排列，内容密度适中，视觉焦点清晰
